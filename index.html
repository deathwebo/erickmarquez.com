My Experience

Execute below query to retrieve detailed information about my experience

        
        SELECT 
            j.position,
            c.name AS company_name,
            date_format(j.from_date, "%b %Y"), 
            if(j.present, "Present", date_format(j.to_date, "%b %Y")),
            GROUP_CONCAT(s.name) AS skills
        FROM jobs AS j
        LEFT JOIN companies AS c ON j.company_id = c.id
        LEFT JOIN skills AS s.job_id = j.id
        GROUP BY j.id;
